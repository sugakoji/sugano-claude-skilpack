# sugakoji Claude Plugin

個人用の Claude Code プラグイン。

## セットアップ

Claude Code 内で以下を実行：

```
/plugin marketplace add sugakoji/sugakoji-claude-skilpack
/plugin install sugakoji@sugakoji-plugins
```

## スキル一覧

| コマンド | 説明 | 何が便利か |
|---------|------|-----------|
| `/sugakoji:open-vscode-browser` | VSCode内のBrowse Liteで指定URLを開く | URL指定や自然言語で、VSCodeから離れずにGitHub PRやlocalhostをその場で確認できる |

## アンインストール

Claude Code 内で以下を実行：

```
/plugin uninstall sugakoji@sugakoji-plugins
```
