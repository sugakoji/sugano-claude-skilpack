# sugano Claude Plugin

個人用の Claude Code プラグイン。

## セットアップ

Claude Code 内で以下を実行：

```
/plugin marketplace add sugakoji/sugano-claude-skilpack
/plugin install sugano@sugano-plugins
```

## スキル一覧

| コマンド | 説明 | 何が便利か |
|---------|------|-----------|
| `/sugano:open-vscode-browser` | VSCode内のBrowse Liteで指定URLを開く | URL指定や自然言語で、VSCodeから離れずにGitHub PRやlocalhostをその場で確認できる |

## アンインストール

Claude Code 内で以下を実行：

```
/plugin uninstall sugano@sugano-plugins
```
